# Python project template

## File structure

.
├── README.md
├── source
|   ├── app.py
|   ├── setup.py
    ├── requirements.txt
|   ├── modules
|   ├── config
|   |    └── logging.yml
|   └── logs
└── .gitignore

## Functionality included

- Logging

## Work inside source folder

- Enter `cd source`

## Run application

- Do `python app.py`

## Use venv if python3 is not default

- Do `python3 -m venv myenv`
- Do `source myenv/bin/activate`
